# Git Todo Monitor - Roadmap

This document outlines the planned features and improvements for Git Todo Monitor.

## Current Version: 1.0.0

✅ Core git monitoring and analysis  
✅ AI-powered summarization  
✅ Desktop notifications  
✅ Hourly cron scheduling  
✅ Commit hook triggering  
✅ Performance optimizations  

---

## Next Steps (v1.1.0) - Q1 2025

### Enhanced Notification Options
- [ ] **Email notifications** - Send summaries via email
- [ ] **Slack integration** - Post summaries to Slack channels
- [ ] **Discord webhooks** - Send summaries to Discord servers
- [ ] **Telegram bot** - Deliver summaries via Telegram

### Multi-Repository Support
- [ ] **Monitor multiple repos** - Track several repositories from one config
- [ ] **Consolidated summaries** - Single notification for all repos
- [ ] **Per-repo configuration** - Different settings for each repository

### Customization Improvements
- [ ] **Custom todo patterns** - Define your own comment markers
- [ ] **Priority system** - Flag high-priority todos (e.g., URGENT, CRITICAL)
- [ ] **Filtering options** - Exclude specific files/directories
- [ ] **Verbosity levels** - Brief, normal, or detailed summaries

---

## Future Enhancements (v2.0.0) - Q2 2025

### Advanced Features
- [ ] **Web dashboard** - View summary history in browser
- [ ] **Todo analytics** - Track todo trends over time
- [ ] **GitHub/GitLab integration** - Monitor remote repositories via API
- [ ] **Team features** - Share summaries with team members
- [ ] **Smart scheduling** - Adjust frequency based on commit activity

### AI Enhancements
- [ ] **Multiple AI providers** - Support for Anthropic, local LLMs
- [ ] **Context-aware summaries** - Learn from past summaries
- [ ] **Code review insights** - Suggest improvements beyond todos
- [ ] **Automated todo categorization** - Group by type, urgency, file

### Developer Experience
- [ ] **VS Code extension** - In-editor notifications
- [ ] **CLI improvements** - Interactive mode, better output formatting
- [ ] **Docker support** - Containerized deployment
- [ ] **Configuration wizard** - Interactive setup tool

---

## Long-term Vision (v3.0.0+)

### Intelligence & Automation
- [ ] **Predictive analysis** - Suggest todos based on code patterns
- [ ] **Auto-task creation** - Create tickets in Jira/Linear/GitHub Issues
- [ ] **Code quality scoring** - Track overall project health
- [ ] **Dependency tracking** - Link todos to dependencies

### Collaboration
- [ ] **Team dashboards** - Shared visibility across teams
- [ ] **Todo assignment** - Auto-assign based on code ownership
- [ ] **Progress tracking** - Monitor todo completion rates
- [ ] **Integration marketplace** - Community-built extensions
